# Cascade Agent Rules

## Development Server Management
1. **NEVER automatically start development servers (e.g., `npm run dev`, `yarn dev`, `next dev`).**
2. Assume the development server is already running locally unless the user explicitly asks you to start it.
3. This saves time and prevents resource conflicts in the local environment.

## Design & UI Preferences
1. **Always use Light Mode.** The user explicitly DOES NOT want dark mode or dark backgrounds on the InfraSpine website. Keep all backgrounds and components light and clean.
2. Use modern, premium SaaS designs (e.g., glowing bento boxes, subtle shadows, clean typography).
3. Ensure UI components are appropriately sized and not too massive on screen (compact but readable).
